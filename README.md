 Advanced Packet Sniffer

Real-time network traffic analyzer and threat detection tool built with Python, Flask, and Scapy.

 Features
- Live packet capture — TCP, UDP, ICMP, ARP, DNS
- Port scan and threat detection
- Protocol statistics with live charts
- Web-based dashboard (browser UI)
- BPF filter support

Technologies
Python, Flask, Flask-SocketIO, Scapy, HTML/CSS/JS


 Setup

```bash
git clone https://github.com/tabasumshafi046-cmyk/advanced-packet-sniffer.git
cd advanced-packet-sniffer
python -m pip install -r requirements.txt

 Windows
python app.py

 Linux
sudo python3 app.py
```

Open browser: `http://localhost:5000`
